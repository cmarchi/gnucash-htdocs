6.6. Editing a Transaction

Editing a transaction involves just selecting the part of the transaction that needs to be changed. Once the changes are complete pressing the Enter key, selecting the Enter icon or going to Transaction -> Enter Transaction will now move the cursor to the next line or transaction.

To see a more detailed view of a transaction for editing in Basic Ledger, press the Split button on the toolbar or select Actions -> Split Transaction .
