5.5. Modificare un conto

La finestra di modifica del conto consta di una sola linguetta per le informazioni generali. Questa finestra presenta le stesse informazioni che sono già state descritte in come creare un nuovo conto Sezione 5.4. Per accedere alla finestra di modifica del conto andare in Modifica -> Modifica conto (scorciatoia di menu Ctrl-E) o premere l'icona della barra strumenti Modifica nella finestra della visualizzazione ad albero dei conti.
