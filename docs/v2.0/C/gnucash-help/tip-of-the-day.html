3.2. Tip of The Day

The Tip of the Day screen starts whenever you start GnuCash unless it has been disabled. It gives tips on features and using GnuCash. You can use the three buttons at the bottom of the screen to look through the tips.

  • The Close button is used to dismiss the Tip of the Day dialog

  • The Prev button is used to look at the previous tip

  • The Next button is used to look at the next tip.

  • The Display this dialog next time check-box is used to enable or disable the Tip of the Day from running at GnuCash start-up.

It is also possible to configure the Tip of the Day to run at start-up in Edit -> Preferences -> General. The Tip of the Day can also be manually run by going to Help -> Tip of the Day.
