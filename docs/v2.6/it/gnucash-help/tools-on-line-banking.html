7.5. Procedura guidata per l’HBCI (Online Banking)

Nota

Questa sezione è "in costruzione - ogni contributo è benvenuto!!".

La procedura guidata per l’HBCI permette di creare e modificare i dati necessari per accedere alle transazioni registarte dall’online banking. Attualmente, le istruzioni più complete per questo procedimento sono nella wiki di GnuCash all’indirizzo Setting up OFXDirectConnect in GnuCash 2. Se la voce Impostazione dell’Online Banking non appare nel proprio menu Strumenti, verificare di essere nella finestra della struttura dei conti o nella scheda del registro.
