Introdução

O GnuCash é uma aplicação que serve para controlar as suas finanças. Para poder começar desde já a registar as suas finanças, veja o Guia de Iniciação Rápida. Para mais informação sobre as muitas funcionalidades que o GnuCash oferece, consulte a página das Funcionalidades.
