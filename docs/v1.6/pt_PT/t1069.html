A janela de nova conta terá este aspecto:

Escolha um Tipo de Conta.

O campo de Moeda deverá tipicamente ser um código ISO de moeda de três letras (por ex. - PTE para Escudos Portugueses). O Código de Conta é um número que determina a ordenação da conta quando ela surgir num relatório ou no Plano de Contas.

A imagem abaixo representa um exemplo de uma conta de negociação de acções ou moedas. Note que o campo Activo não está desabilitado e que pode introduzir um valor. Esse valor deverá ser tipicamente um código de moeda ISO de três letras.
