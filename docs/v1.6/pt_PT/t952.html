Janela Final

Quando um utilizador inicia o GnuCash pela primeira vez ou pede no menu "Ficheiro" para criar um "Novo Ficheiro" este druida é executado para dar apoio na criação de uma lista de contas. Você pode desligar o druida que surge ao criar novo ficheiro activando a a opção "Sem configuração de lista de contas em novo ficheiro" na secção "Genérica" da janela Preferências .

O druida será executado para um novo utilizador até que este utilize um ficheiro, ou complete o druida, ou remova a selecção na caixa que surge quando é primido o "Cancelar".

Você pode primir Cancelar a qualquer momento para parar o druida. Uma vez tendo terminado um determinado passo prima o botão Seguinte para passar ao passo seguinte. Caso se aperceba que cometeu um erro ou se esqueceu de algo pode primir o botão Anterior para ser levado até ao passo anterior.
