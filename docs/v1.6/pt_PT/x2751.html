Utilizar o PostGres para armzenar os seus dados

CORRIGIR: Necessita de Conteúdo!

O GnuCash tem suporte beta para a utilização do postgres, um motor de base de dados poderoso, livre, quase totalmente de acordo com o standard SQL, como forma de armazenar os dados em vez de se recorrer à utilização de ficheiros.

Neste momento, a única documentação encontra-se no ficheiro src/doc/engine/sql/README na distribuição de código fonte do GnuCash.
