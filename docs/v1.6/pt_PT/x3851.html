Formatos de data personalizados

Caso deseje utilizar outro formato de data para além dos disponíveis na lista, seleccione Personalizar e introduza um formato de data na caixa Formato de Data na área de definições personalizadas. Se o formato de data personalizado for seleccionado, este texto de formatação é passado para o strftime(3) como um identificador de formato.
