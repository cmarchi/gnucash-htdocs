O Menu Ficheiro

Tal como a maioria das aplicações, as operações com ficheiros são feitas através do menu ficheiro:

  • Novo Ficheiro: Crie um ficheiro GnuCash novo e vazio. Se tiver trabalho por gravar, o GnuCash irá perguntar-lhe se o deseja gravar.

  • Abrir: Carregue um ficheiro GnuCash existente.

  • Gravar: Grave o seu ficheiro de dados GnuCash actual para disco com o seu nome actual.

  • Gravar como: Grave o seu ficheiro de dados GnuCash actual com um novo nome.

  • Exportar Contas : Exporte a sua estrutura de contas, sem quaisquer transacções. Útil para criar bases de trabalho para outras pessoas, ou como uma forma simples, mas ainda assim útil, de fechar os livros (encerrar um período).

  • Importar QIF: Importar ficheiros de dados QIF .

  • Nova Árvore de Contas: Um atalho para criar muitas das contas comuns utilizadas por um agregado familiar.

  • Atalhos para ficheiros carregados anteriormente: os últimos quatro ficheiros GnuCash que você abriu irão surgir. Seleccione-os para os carregar.

  • Mover para Nova Janela: crie uma nova janela de topo. Veja aqui para mais detalhes.

  • Fechar Esta Janela: Fecha esta janela de topo, deixando as restantes abertas. Veja aqui para mais detalhes.

  • Sair do GnuCash: Fecha todas as janelas e sai. O GnuCash irá questiona-lo para gravar o ficheiro, caso tenham ocorrido modificações desde a última gravação.
