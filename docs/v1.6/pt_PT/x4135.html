Importar dados de outras aplicações de finanças pessoais

Se tiver utilizado outra aplicação financeira pessoal que possa exportar ficheiros QIF, tal como o Quicken ou o Microsoft Money, pode importar os dados para o GnuCash. Veja as instruções para o importador QIF.

Uma vez tendo feito isso, está tudo pronto para começar - mas primeiro, um par de dicas rápidas sobre coisas que o GnuCash faz de um modo ligeiramente diferente dessas aplicações:

  • Onde algumas aplicações utilizam "categorias" para catalogar os tipos de despesas e receitas, o GnuCash utiliza contas de despesas e receitas. Elas funcionam de forma idêntica, mas é uma forma consideravelmente mais correcta e flexível de gerir as coisas, uma vez ocê se habituando à ideia.

  • Algumas aplicações utilizam uma lista de contas. O GnuCash utiiza uma "árvore" de contas, permitindo-lhe agrupar conjuntos semelhantes de contas - muito útil quando se geram relatórios dos dados financeiros. Pode rearranjar as suas contas para utilizar uma estrutura mais tipo-árvore utilizando a Janela de Edição de Contas.
