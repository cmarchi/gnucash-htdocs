GnuCash como um Browser Web?

Table of Contents

Necessária uma Imagem!!!

O GnuCash tem a capacidade de apresentar a maioria das páginas web como um relatório "Endereço Frame". Isto pode ser muito útil, particularmente como uma forma de apresentar informação financeira baseada na web, combinada com dados gnucash privados como parte de uma Visualização em Multi-coluna.

Note que o visualizador web do GnuCash não é tão sofisticado como os browsers web dedicados, e poderá não apresentar total ou correctamente páginas que contenham Javascript, animações em Flash, filmes, som, e por aí fora. Se deseja ver páginas assim, necessita de um browser dedicado como o Mozilla.

Opções suportadas por este relatório incluem os seguintes pontos:

  • Título de Relatório

  • Folhas de Estilo (Note: esta opção praticamente não tem efeito neste relatório em específico)

  • Endereço: O endereço a visualizar.
