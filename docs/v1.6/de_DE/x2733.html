Selecting MDI Modes

MDI mode can be set in two different ways. Firstly you can set it within GnuCash through the "Application MDI Mode" option in the Preferences Dialog

However, if you prefer a particular MDI mode across all your GNOME applications, you can set the preference for all of them in the MDI section of the GNOME Configuration program (see the GNOME on-line help for more details). To get GnuCash to recognize the global setting, set the "Application MDI Mode" option in the preferences dialog to "Use GNOME Default".
