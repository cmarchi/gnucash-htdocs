The Main Menu

The main menu is displayed above all top-level windows, and is a gateway to everything that GnuCash can do, including file operations, reports, account operations, window operations, and getting help. It is dynamic, changing to provide the functions required depending on whether you are displaying a report, or an account tree.

The menus that are always present include:

  • File: Items in this menu to create, import, load, and save accounts. See File Operations.

  • Reports

  • Tools: this menu contains the General Ledger, as well as tools to

    • edit "meta-information" about currencies, mutual funds, and stocks,

    • record stock and currency values,

    • Do financial calculations, and

    • Find Transactions.

    • Settings: Customize Style Sheets and Preferences.

    • Help
