Duplicate Transactions

Ken has also written a script that removes duplicate transactions at the QIF level. GnuCash is very strict about duplicates, and this script is useful for successfully importing a succession of QIF files provided by a bank for overlapping time periods, for instance. This script is available from:

http://www-cad.eecs.berkeley.edu/~gooch/qifuniq.html
