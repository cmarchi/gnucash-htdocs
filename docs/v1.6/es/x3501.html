La Ventana de Listados

Una vez ha seleccionado un listado, aparece la ventana de listados:

El listado es mostrado en una ventana de tamaño variable y con desplazamiento. Para controlar el listado, puede utilizar los botones de la barra de herramientas de la parte superior de la ventana:

  • Atrás y Siguiente le permite visualizar el contenido previo de la ventan, como un navegador web

  • Parámetros despliega un diálogo que le permite modificar la información que se presenta en el listado:

    Vea la ayuda de cada listado individual para la explicación de los opciones disponibles.

  • Exportar le permite exportar el contenido de un listado como un archivo HTML para ser visualizado o imprimido posteriormente.

  • Cerrar cierra la ventana de listados.
