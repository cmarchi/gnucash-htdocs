About DocBook

The GnuCash documentation has been changed to use SGML, and, in particular, the DocBook DTD.

DocBook is a DTD designed for authoring technical documents such as computer manuals. A number of tools are growing up to assist in processing DocBook-based material.

    Adds tags for DocBook to support object oriented languages like Java, C++, and IDL, including such new entities as classes, fields, methods, exceptions, and so forth.

  • Get Going With DocBook

    This document describes how one might use the DocBook DTD to write documents, including information on how to use tables, and insert program listings.
