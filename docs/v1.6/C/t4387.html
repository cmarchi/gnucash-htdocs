Register Keyboard Shortcuts

Table of Contents

The following keys can be used to move around in the register: Please check that I didn't miss any

  • Right Arrow, Left Arrowmove left and right within one field

  • Home move to the left of the field

  • End move to the right of the field

  • TAB move to the next field

  • Shift + TAB move to the previous field

  • Enter, Down Arrow move to the next transaction (if necessary, a new blank transaction will be created)

  • Up Arrow move to the previous transaction

  • Page Up move up one screenful

  • Page Down move down one screenful

  • Shift + Page UpGo to the first transaction

  • Shift + Page DownGo to the blank (last) transaction
