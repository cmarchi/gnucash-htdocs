Getting quotes online

GnuCash is able to get quotes online from a variety of sources, through the Finance::Quote module. To support online quotations for a particular stock or mutual fund account, you must first enable online price quoting and select a price source. You can do this in the edit account dialog box. More details on exactly how to choose a price source can be found in the price sources help entry.

Once that is done, you can update prices for your stocks and mutual funds by clicking the "get quotes" button.

If you wish to update price quotes from the command line, you can do so by the following command

gnucash --add-price-quotes <gnucash-file>

GnuCash 1.4 had a helper script for getting price quotes, but this is no longer needed.
