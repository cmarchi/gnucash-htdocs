5.5. Editing an Account

The Edit Account dialog consists of just the General tab. This tab has the same information that was described in Section 5.4, “Creating a New Account”. To access the Edit Account dialog go to Edit → Edit Account (menu shortcut Ctrl+E) or click the Edit Toolbar icon in the Account Tree Window.
