6.9. Copying a Transaction

Copying Transactions is available from the Transaction menu. The Transaction menu has Cut Transaction, Copy Transaction and Paste Transaction which allows a whole transaction to be cut or copied and pasted to a new transaction line. This way of copying a transaction will use the date showing in the new transaction blank line.

The Transaction → Duplicate Transaction... or the Toolbar Duplicate which, unlike the Copy method, allows for choosing a different date.

  • Select the transaction to duplicate.

  • Go to Transaction → Duplicate Transaction...

  • A dialog called Duplicate Transaction will prompt for a new Date and Num for the transaction.

  • Press OK to add the transaction to the register.
