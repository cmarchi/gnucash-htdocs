6.6. Modificare una transazione

Per modificare una transazione è sufficiente selezionare la parte della transazione stessa che si desidera modificare. Una volta effettuati i cambiamenti, premere il tasto Invio, oppure selezionare il pulsante Inserisci o andare in Transazione → Inserisci Transazione per spostare il cursore nella riga successiva della transazione.

Per visualizzare maggiori dettagli relativi a una transazione nella modalità a libro mastro basilare, premere il pulsante Suddivisione nella barra degli strumenti o selezionare Operazioni → Suddividi transazione.
